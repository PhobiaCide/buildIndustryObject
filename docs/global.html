getData_

Fetches the given Eve Online Static Data Export .JSON conversion and returns the parsed JSON

Takes as input a table name, a table address, and a toggle for using the cache service. If a table name and a table address are boh provided, the script will use the address. The response is parsed as JSON on the return.
Properties:
Name Type Description
options.tableName string The name of the desired .JSON conversion file, defaults to {boolean} false
options.tableAddress string The address of the desired .JSON conversion file, has preference over options.tableName, defaults to {boolean} false
options.useCache boolean Weather to use the cache service or not
